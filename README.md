💰 SmartSpend – Expense Management System
SmartSpend is a full-stack web application that helps users track, manage, and analyze their daily expenses efficiently. It provides a simple and intuitive interface to monitor spending habits and stay financially organized.

🚀 Features:-
🔐 User Authentication (Signup & Login)
➕ Add Expenses
📜 View Expense History
❌ Delete Expenses
📊 Dashboard with total spending overview
💡 Clean and responsive UI

🛠️ Tech Stack:-
Frontend:
HTML
CSS
JavaScript
Backend:
Node.js
Express.js
Database:
MongoDB Atlas

📁 Project Structure:-
WEBTECH/
│
├── models/           # Mongoose models (User, Expense)
├── public/           # Frontend files (HTML, CSS, JS)
├── node_modules/

├── package.json
